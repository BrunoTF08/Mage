# Mage

Ghost castle game third person

## Ghost combat feedback

- Right mouse button locks aim onto the nearest visible ghost.
- Locked ghosts show a white outline aura instead of a HUD warning text.
- The ghost health bar appears only while the ghost is locked.
- Ghost health bars use an instant fill drop plus a delayed trailing bar, matching the player health feedback.
- Ghosts play their death animation and are destroyed when health reaches zero.
